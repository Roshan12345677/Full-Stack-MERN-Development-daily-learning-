## 🔹 1. PRD (Product Requirement Document)

PRD defines the complete plan of a project.

### 📌 Key Sections:

* Objective
* Scope
* Features & Requirements (P1, P2)
* User Stories
* Technical Requirements
* Design Requirements
* Success Metrics
* Timeline

---

### 🧃 Example: Juice Shop PRD

#### 🎯 Objective

Build an HTML website for a juice shop and convert offline shop to online presence.

#### 📦 Scope

* Basic website with product cards
* Contact details
* ❌ No payment system (for now)

#### ⭐ Features

**P1 (Mandatory):**

* Juice images
* Price display

**P2 (Optional):**

* Pay with Bitcoin

---

### 👤 User Stories

* User wants to order juice online
* User wants secure web-based system

---

### ⚙️ Technical Requirements

* HTML
* CSS
* JavaScript

---

### 🎨 Design Requirements

* Clean & natural UI
* Animated website
* Light & dark mode

---

### 📊 Timeline

* 1 Month

## 🔹 2. HTML (HyperText Markup Language)

HTML is used to create structure of web pages.

👉 Think:

* HTML → Structure
* CSS → Styling
* JS → Logic

---

## 🔹 3. Basic HTML Structure

```html id="htmlbasic"
<!DOCTYPE html>
<html>
<head>
    <title>My Page</title>
</head>
<body>
    <h1>Hello World</h1>
    <p>This is a paragraph</p>
</body>
</html>
```

---

## 🔹 4. Important HTML Tags

### 📌 Text & Structure

* `<h1>` to `<h6>` → Headings
* `<p>` → Paragraph
* `<div>` → Block container
* `<span>` → Inline container

---

### 🔗 Links & Media

```html id="linkimg"
<a href="https://google.com">Click Me</a>
<img src="image.jpg" alt="image">
```

---

### 📝 Lists

```html id="lists"
<ul>
  <li>Home</li>
  <li>About</li>
</ul>

<ol>
  <li>Step 1</li>
  <li>Step 2</li>
</ol>
```

---

### 📊 Tables

```html id="table"
<table>
  <tr>
    <th>Name</th>
    <th>Price</th>
  </tr>
  <tr>
    <td>Mango Juice</td>
    <td>₹50</td>
  </tr>
</table>
```

---

### 📥 Forms

```html id="form"
<form>
  <input type="text" placeholder="Enter name">
  <input type="submit">
</form>
```

---

### 🎥 Video

```html id="video"
<video src="video.mp4" controls></video>
```

---

## 🔹 5. Semantic Tags (Layout)

* `<header>` → Top section
* `<main>` → Main content
* `<section>` → Section
* `<article>` → Independent content
* `<aside>` → Side content
* `<footer>` → Bottom

---

## 🔹 6. Special Tags

* `<br>` → Line break
* `<hr>` → Horizontal line
* `<pre>` → Preserve formatting

---

## 🔹 7. HTML Concepts

* HTML is **case-insensitive**
* Tags = Opening + Closing
* Some tags are self-closing (`<br>`, `<hr>`)

---

## 🔹 8. SEO (Search Engine Optimization)

Helps website rank better on search engines.

### 📌 Techniques:

* Use proper title
* Add meta description

```html id="meta"
<meta name="description" content="Juice shop website">
```

* Use meta author
* Add favicon
* Optimize images
* Remove unused code
* Add alt text to images

---

## 🧠 Key Learnings

* PRD helps in planning projects
* HTML builds structure of web pages
* Semantic tags improve readability
* SEO improves website visibility
